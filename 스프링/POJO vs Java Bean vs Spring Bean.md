## POJO
모든 자바 객체
- 객체 지향적인 원리에 충실하면서 환경과 기술에 종속되지 않고, 필요에 따라 재활용될 수 있는 방식으로 설계된 오브젝트를 의미

<br>

## Java Bean
데이터를 표현하는 것을 목적으로 하는 자바 클래스
> **충족요건**
> 1. 기본 생성자가 있어야 한다.(no-arg 생성자)
> 2. getter, setter가 있어야한다.
> 3. 직렬화 기능(Serializable 인터페이스 사용)

<br>
